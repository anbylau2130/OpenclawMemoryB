## 📖 使用建议

### 1. 技能选择原则
- **先读 SKILL.md** - 每个技能都有详细文档
- **匹配触发词** - 使用正确的关键词触发
- **按需使用** - 不是所有任务都需要技能

### 2. 技能组合使用
```
创意工作流程：
brains → frontend-design → 开发实现

小说创作流程：
novel-worldbuilding → novel-research → novel-prose 
→ novel-review → novel-archiving → novel-memory

项目开发流程：
TaskMaster → ralph-loop → code-review → github
```

### 3. 技能文件位置
```
skills/
├── brains/
│   └── SKILL.md
├── frontend-design/
│   └── SKILL.md
├── skill-creator/
│   └── SKILL.md
├── TaskMaster/
│   └── SKILL.md
├── ...
└── weather/
    └── SKILL.md
```

### 4. 查看技能详情
```bash
# 查看单个技能
cat skills/brains/SKILL.md

# 列出所有技能
ls -1 skills/

# 搜索技能关键词
grep -r "关键词" skills/*/SKILL.md
```
