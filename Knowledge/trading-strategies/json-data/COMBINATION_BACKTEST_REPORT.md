## 💡 策略组合原理

### 1. 投票法（推荐）
```
买入信号：RSI买入 AND 布林带买入
卖出信号：RSI卖出 AND 布林带卖出
```

**优势：**
- 双重确认，降低误判
- 提高胜率
- 稳定收益

**劣势：**
- 交易次数减少
- 可能错过部分机会

### 2. 加权法
```
买入信号：(RSI权重*0.5 + 布林带权重*0.3 + 双均线权重*0.2) > 0.5
```

**优势：**
- 灵活调整权重
- 重视高胜率策略

**劣势：**
- 参数调优复杂
- 当前测试表现不佳

### 3. 顺序法
```
第一步：RSI确认趋势
第二步：布林带确认入场点
```

**优势：**
- 逻辑清晰
- 趋势+位置双重确认

**劣势：**
- 条件过于严格
- 交易机会少

## 🔍 深度分析

### 为什么震荡市组合表现最佳？

**1. 策略互补性**
- **RSI：** 判断超买超卖（动量维度）
- **布林带：** 判断价格位置（波动率维度）
- 两者从不同角度确认信号

**2. 市场环境匹配**
- 当前市场：下跌+震荡
- RSI：捕捉超卖反弹
- 布林带：捕捉价格回归

**3. 条件适度**
- 双策略投票：不太严格，也不太宽松
- 三策略投票：过于严格，错失机会
- 单一策略：误判率较高

### 为什么其他组合表现不佳？

**1. 趋势市组合（双均线+MACD）**
- 市场环境不匹配（下跌市）
- 趋势策略在震荡市表现差

**2. 三策略组合**
- 条件过于严格
- 交易机会少（仅1次）
- 误判一次影响大

**3. 加权法和顺序法**
- 参数调优不足
- 当前参数不适合模拟数据
