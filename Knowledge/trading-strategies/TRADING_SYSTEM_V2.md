## 🏗️ 系统架构

### 设计理念

```
核心理念：多策略验证 + 量化评分 + 仓位管理

优势：
✅ 降低单一策略失败风险
✅ 客观量化，避免情绪干扰
✅ 动态仓位，控制风险
✅ 规则明确，易于执行
```

```

                  ↓
```

## 📊 策略评分体系

### 迭代1：基础评分框架

#### 评分公式

```
综合胜率 = 基础胜率(50%) + Σ(策略信号 × 权重)

计算步骤：
1. 确定基础胜率：50%（中性）
2. 计算每个策略的信号得分
3. 乘以对应权重
4. 汇总得到综合胜率
5. 根据胜率等级生成信号
```

#### 策略分类与权重

| 类别 | 基础胜率 | 权重 | 包含策略 |
|------|---------|------|---------|
| **趋势跟踪** | 60% | 30% | MACD、MA系统、趋势强度 |
| **均值回归** | 65% | 25% | RSI、Bollinger、KDJ |
| **突破交易** | 70% | 20% | Dual Thrust、区间突破 |
| **量价分析** | 60% | 15% | 成交量配合、量价背离 |
| **技术指标** | 55% | 10% | OBV、形态识别 |

```python
突破20日高点 = +12% × 20% = +2.4%

# 量价类（权重15%）
价涨量增 = +10% × 15% = +1.5%

# 技术指标类（权重10%）
OBV上升 = +5% × 10% = +0.5%

# 总分
综合胜率 = 50% + 4.5% + 3.0% + 1.5% + 2.0% + 2.4% + 1.5% + 0.5%
         = 65.4%
```

### 迭代5：动态仓位系统

#### 总仓位控制矩阵

| 大盘环境 | 判断标准 | 最大仓位 | 单只最大 | 说明 |
|---------|---------|---------|---------|------|
| **超级牛市** | 指数在MA60上方+连续5日上涨 | 90% | 40% | 激进 |
| **牛市** | 指数在MA60上方+连续上涨 | 80% | 35% | 积极 |
| **偏强震荡** | 指数在MA20-MA60之间 | 60% | 30% | 适度 |
| **震荡市** | 指数在MA20附近 | 50% | 25% | 保守 |
| **偏弱震荡** | 指数在MA20-MA60下方 | 40% | 20% | 谨慎 |
| **熊市** | 指数在MA60下方+连续下跌 | 30% | 15% | 防守 |
| **超级熊市** | 指数跌破重要支撑+恐慌 | 20% | 10% | 极度谨慎 |

#### 单只股票仓位公式

```
单只仓位 = 总仓位 × min((综合胜率 - 50%) / 25%, 1) × 0.5

公式说明：
- 基础仓位：总仓位的50%
- 评分加成：根据综合胜率调整
- 上限控制：单只不超过总仓位的50%

示例（总仓位50%）：
- 胜率80%：50% × min(30%/25%, 1) × 0.5 = 50% × 1 × 0.5 = 25%
- 胜率75%：50% × min(25%/25%, 1) × 0.5 = 50% × 1 × 0.5 = 25%
- 胜率70%：50% × min(20%/25%, 1) × 0.5 = 50% × 0.8 × 0.5 = 20%
- 胜率65%：50% × min(15%/25%, 1) × 0.5 = 50% × 0.6 × 0.5 = 15%
- 胜率60%：50% × min(10%/25%, 1) × 0.5 = 50% × 0.4 × 0.5 = 10%
```

| **75-79%** | 30% | 盈利3%加25% | 盈利6%加15% | 60% |

#### 金字塔加仓法

```
示例：10万本金，总仓位50%，单只最大25%

第1次建仓（胜率75%）：
- 买入金额：10万 × 25% = 2.5万
- 价格：20.00
- 数量：1250股

第2次加仓（盈利3%，价格20.60）：
- 加仓金额：2.5万 × 80% = 2万
- 价格：20.60
- 数量：970股

第3次加仓（盈利6%，价格21.20）：
- 加仓金额：2.5万 × 50% = 1.25万
- 价格：21.20
- 数量：590股

总持仓：
- 总数量：1250 + 970 + 590 = 2810股
- 总成本：(25000 + 20000 + 12500) / 2810 = 20.47
- 当前价：21.20
- 浮盈：(21.20 - 20.47) / 20.47 = 3.57%
- 持仓比例：(25000 + 20000 + 12500) / 100000 = 57.5%
```

```
辅助条件（满足2个以上）：
✅ 成交量正常

✅ 均线配合

操作：
- 次日建仓20%
```

**轻仓买入（65-69%）**

```
必要条件：
✅ 综合评分 65-69%

操作：
- 小仓位试探15%
- 严格止损：-3%
- 不主动加仓
```

#### 加仓信号

```
条件1（满足3个以上）：
✅ 持仓盈利 > 3%
✅ 综合评分上升（比买入时高5%以上）
✅ 突破前高
✅ 成交量放大（量比 > 1.5）
✅ 新增2个以上买入信号

条件2（趋势确认）：
✅ 价格在MA5上方
✅ 连续2日上涨
✅ MACD保持多头

操作：
- 加仓幅度 = 首次建仓 × 80%
- 更新止损位（保本或盈利2%）
```

#### 减仓信号

```
条件1（满足2个以上）：
⚠️ 综合评分下降 > 10%
⚠️ 出现2个以上卖出信号
⚠️ 量价背离
⚠️ 接近止盈目标（盈利8%以上）

条件2（趋势转弱）：
⚠️ 价格跌破MA5
⚠️ 成交量异常（量比 > 3或< 0.5）
⚠️ 连续2日下跌

操作：
- 减仓50%
- 剩余仓位提高止损
- 重新评估
```

#### 卖出信号

**止盈卖出**

```
分级止盈：
✅ 盈利10% → 卖出50%，保留50%
✅ 盈利15% → 再卖出25%，保留25%
✅ 盈利20% → 全部卖出

条件止盈：
✅ 综合评分 < 60% → 全部卖出
✅ 出现3个以上卖出信号 → 全部卖出
✅ MACD死叉 → 全部卖出

执行原则：
- 到达目标立即执行
- 不贪心，落袋为安
```

**止损卖出**

```
硬止损（不可动摇）：
❌ 亏损达到5% → 立即全部卖出
❌ 跌破MA20 → 卖出50%
❌ 综合评分 < 55% → 全部卖出

软止损（灵活执行）：
⚠️ 亏损3% → 观察并准备止损
⚠️ 跌破MA10 → 卖出50%
⚠️ 连续3日下跌 → 卖出50%
⚠️ 成交量持续萎缩 → 减仓

执行原则：
- 硬止损无条件执行
- 软止损结合市场环境
- 止损后冷静期2天
```

## ⚠️ 风险管理

### 迭代8：全面风控体系

#### 单笔风险控制

```
单笔最大亏损 = 总资金 × 2%
单笔最大持仓 = 总资金 × 40%

止损位计算：
止损位 = 买入价 × (1 - 单笔最大亏损 / 单笔最大持仓)
       = 买入价 × (1 - 2% / 40%)
       = 买入价 × 0.95

示例：
买入价：20.00
止损位：20.00 × 0.95 = 19.00
最大亏损：(20.00 - 19.00) × 持仓数量 / 总资金 = 2%
```

#### 总体风险控制

| 风险类型 | 限制 | 触发后操作 |
|---------|------|-----------|
| **单日最大亏损** | 3% | 停止交易，冷静1天 |
| **单周最大亏损** | 5% | 停止交易，冷静3天 |
| **单月最大亏损** | 10% | 停止交易，重新评估 |
| **连续亏损次数** | 3次 | 减半仓位，冷静期 |
| **单笔最大回撤** | 8% | 强制止损 |

#### 回撤控制

| 回撤幅度 | 风险等级 | 操作 |
|---------|---------|------|
| **0-5%** | 🟢 正常 | 继续操作 |
| **5-10%** | 🟡 警惕 | 减仓50%，提高止损 |
| **10-15%** | 🟠 危险 | 清仓50%，暂停新开仓 |
| **>15%** | 🔴 紧急 | 全部清仓，停止交易1周 |

#### 市场风险应对

**系统性风险**

```
识别信号：
⚠️ 大盘连续3日大跌
⚠️ 北向资金连续流出
⚠️ 市场恐慌情绪蔓延
⚠️ 政策面重大利空

应对措施：
1. 立即减仓至30%以下
2. 提高所有持仓止损位
3. 暂停新开仓
4. 等待市场企稳
```

**流动性风险**

```
识别信号：
⚠️ 成交量突然萎缩
⚠️ 买卖价差扩大
⚠️ 涨跌停板频繁

应对措施：
1. 降低仓位
2. 避免追涨杀跌
3. 使用限价单
4. 分批交易
```

#### 黑天鹅应对

```
预案：
1. 立即评估影响程度
2. 如果影响重大（>10%波动）：
   - 立即全部清仓
   - 等待事件明朗
3. 如果影响有限：
   - 减仓50%
   - 提高止损
   - 密切观察

恢复：
- 事件平息后
- 市场企稳后
- 小仓位试探
- 逐步恢复正常
```

```
4. 控制情绪：按规则操作，不受情绪影响
5. 保持耐心：机会总会出现，不急躁
6. 承认错误：止损后不纠结，向前看
7. 持续学习：每次交易都是学习机会
8. 记录总结：详细记录每笔交易
```
