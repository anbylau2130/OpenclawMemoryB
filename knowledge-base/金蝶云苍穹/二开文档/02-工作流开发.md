## 📋 概述

工作流（Workflow）是金蝶云苍穹的核心功能，用于定义和管理业务审批流程、自动化任务和跨系统协作。

## 🔧 核心概念

### 1. 流程定义

**流程定义示例（C#）**:
```csharp
using K3Cloud.Workflow.Engine;

public class CustomApprovalWorkflow
{
    public string WorkflowId { get; set; } = "kdev_custom_approval";
    public string WorkflowName { get; set; } = "自定义审批流程";
    public string Version { get; set; } = "1.0";
    public WorkflowType Type { get; set; } = WorkflowType.Approval;
    
    public List<WorkflowNode> Nodes { get; set; }
    public List<WorkflowEdge> Edges { get; set; }
    
    public CustomApprovalWorkflow()
    {
        // 初始化流程节点
        Nodes = new List<WorkflowNode>
        {
            new WorkflowNode
            {
                NodeId = "start",
                NodeType = NodeType.Start,
                NodeName = "开始",
                Position = new Position { X = 100, Y = 100 }
            },
            new WorkflowNode
            {
                NodeId = "dept_approval",
                NodeType = NodeType.Approval,
                NodeName = "部门审批",
                Position = new Position { X = 300, Y = 100 },
                Config = new ApprovalConfig
                {
                    ApproverType = ApproverType.DeptHead,
                    Timeout = 7200,
                    AutoApprove = false
                }
            },
            new WorkflowNode
            {
                NodeId = "manager_approval",
                NodeType = NodeType.Approval,
                NodeName = "经理审批",
                Position = new Position { X = 500, Y = 100 },
                Config = new ApprovalConfig
                {
                    ApproverType = ApproverType.Manager,
                    Timeout = 7200,
                    AutoApprove = false
                }
            },
            new WorkflowNode
            {
                NodeId = "end",
                NodeType = NodeType.End,
                NodeName = "结束",
                Position = new Position { X = 700, Y = 100 }
            }
        };
        
        // 初始化流程边
        Edges = new List<WorkflowEdge>
        {
            new WorkflowEdge { Source = "start", Target = "dept_approval" },
            new WorkflowEdge { Source = "dept_approval", Target = "manager_approval", Condition = "approved" },
            new WorkflowEdge { Source = "dept_approval", Target = "end", Condition = "rejected" },
            new WorkflowEdge { Source = "manager_approval", Target = "end" }
        };
    }
}
```

### 2. 审批节点配置

**审批节点类型**:
```javascript
const ApprovalTypes = {
  SINGLE: "single",          // 单人审批
  SEQUENTIAL: "sequential",    // 顺序审批
  PARALLEL: "parallel",        // 并行审批
  OR: "or",                   // 或签（任意一人通过）
  AND: "and"                  // 会签（所有人通过）
};
```

public class MultiApprovalNode

        };
    }
}

### 3. 条件分支

**条件表达式语法**:
```javascript
// 条件表达式
const Conditions = {
  AMOUNT_GT_10000: "${amount} > 10000",      // 金额大于10000
  DEPT_IS_SALES: "${dept} == 'sales'",      // 部门是销售
  STATUS_IS_PENDING: "${status} == 'pending'" // 状态是待处理
};
```

**条件分支示例（JSON）**:
```json
{
  "nodeId": "conditional_node",
  "nodeType": "condition",
  "nodeName": "条件判断",
  "config": {
    "condition": "${amount} > 10000",
    "trueTarget": "manager_approval",
    "falseTarget": "dept_approval"
  }
}
```

**条件分支示例（C#）**:
```csharp
using K3Cloud.Workflow.Expressions;

public class ConditionalNode
{
    public string NodeId { get; set; } = "conditional_node";
    public string NodeType { get; set; } = "condition";
    public string NodeName { get; set; } = "条件判断";
    public ConditionConfig Config { get; set; }
    
    public ConditionalNode()
    {
        Config = new ConditionConfig
        {
            Expression = new Expression("${amount} > 10000"),
            TrueTarget = "manager_approval",
            FalseTarget = "dept_approval"
        };
    }
}
```

### 4. 并行处理

**并行节点配置示例（JSON）**:
```json
{
  "nodeId": "parallel_gateway",
  "nodeType": "parallel",
  "nodeName": "并行处理",
  "config": {
    "branches": [
      {
        "branchId": "branch_1",
        "target": "approval_1"
      },
      {
        "branchId": "branch_2",
        "target": "approval_2"
      },
      {
        "branchId": "branch_3",
        "target": "approval_3"
      }
    ],
    "waitMode": "all"  // all:等待所有分支完成, any:任意一个完成
  }
}
```

**并行节点配置示例（C#）**:
```csharp
public class ParallelGateway
{
    public string NodeId { get; set; } = "parallel_gateway";
    public string NodeType { get; set; } = "parallel";
    public string NodeName { get; set; } = "并行处理";
    public ParallelConfig Config { get; set; }
    
    public ParallelGateway()
    {
        Config = new ParallelConfig
        {
            Branches = new List<Branch>
            {
                new Branch { BranchId = "branch_1", Target = "approval_1" },
                new Branch { BranchId = "branch_2", Target = "approval_2" },
                new Branch { BranchId = "branch_3", Target = "approval_3" }
            },
            WaitMode = WaitMode.All  // 等待所有分支完成
        };
    }
}
```

## 🔄 操作流程

### 1. 创建工作流
```bash
# 使用CLI工具
k3cloud-cli workflow create --file workflow.json
```

### 2. 部署工作流
```javascript
// 使用API部署
POST /api/workflow/deploy
{
  "workflowId": "kdev_custom_approval",
  "version": "1.0",
  "autoActivate": true
}
```

### 3. 启动流程实例
```javascript
// 使用API启动流程
POST /api/workflow/instance/start
{
  "workflowId": "kdev_custom_approval",
  "businessData": {
    "billId": "CUSTOM_BILL_001",
    "amount": 15000,
    "dept": "sales",
    "initiator": "user001"
  }
}
```

### 4. 流程审批
```javascript
// 使用API审批
POST /api/workflow/instance/approve
{
  "instanceId": "WF_INSTANCE_001",
  "nodeId": "dept_approval",
  "action": "approve",
  "comment": "同意"
}
```

## 🎯 最佳实践

### 1. 流程设计
- 保持流程简洁，避免过度复杂
- 合理设置审批节点，避免审批过多
- 使用条件分支提高流程灵活性

### 2. 性能优化
- 使用并行处理提升效率
- 设置合理的超时时间
- 避免循环引用

### 3. 安全考虑
- 审批权限控制
- 敏感操作审计
- 异常监控告警
