## 📋 概述

单据建模是金蝶云苍穹二次开发的核心功能，用于定义和管理业务单据的模板、字段、表单和列表视图。

## 🔧 核心概念

### 1. 单据模板
单据模板是业务单据的基础定义，包含单据的基本属性和元数据。

**示例代码（K3Cloud API）**:
```javascript
// 创建单据模板
{
  "billFormId": "kdev_custom_bill",
  "billFormName": "自定义单据",
  "billFormNumber": "CUSTOM_BILL_001",
  "entityName": "kdev_custom_bill",
  "formType": 1, // 1:单据, 2:基础资料
  "systemType": "K3Cloud"
}
```

**示例代码（C#）**:
```csharp
// C# 重写参考
public class CustomBillTemplate
{
    public string BillFormId { get; set; } = "kdev_custom_bill";
    public string BillFormName { get; set; } = "自定义单据";
    public string BillFormNumber { get; set; } = "CUSTOM_BILL_001";
    public string EntityName { get; set; } = "kdev_custom_bill";
    public int FormType { get; set; } = 1; // 1:单据
    public string SystemType { get; set; } = "K3Cloud";
}
```

## 🔄 操作流程

### 1. 创建单据模板
```bash
# 使用CLI工具
k3cloud-cli bill create --template custom_bill.json
```

### 2. 定义字段
```javascript
// 使用API创建字段
POST /api/bill/fields
{
  "billFormId": "kdev_custom_bill",
  "fields": [
    {
      "fieldId": "kdev_bill_no",
      "fieldName": "单据编号",
      "fieldKey": "bill_no",
      "fieldType": "Text",
      "fieldLength": 50,
      "required": true,
      "unique": true
    }
  ]
}
```

### 3. 设计表单
```javascript
// 使用表单设计器
const formData = await FormDesigner.create({
  billFormId: "kdev_custom_bill",
  layout: formLayout
});
```

### 4. 配置列表视图
```javascript
// 使用列表配置器
const listView = await ListViewDesigner.create({
  billFormId: "kdev_custom_bill",
  columns: columnDefinitions
});
```

## 🎯 最佳实践

### 1. 命名规范
- 单据ID: `kdev_{module}_{entity}`
- 字段ID: `kdev_{entity}_{field}`
- 表名: `T_DEV_{MODULE}_{TABLE}`

### 2. 性能优化
- 合理设置字段长度
- 添加必要的索引
- 使用批量操作

### 3. 安全考虑
- 字段验证
- 权限控制
- 审计日志
