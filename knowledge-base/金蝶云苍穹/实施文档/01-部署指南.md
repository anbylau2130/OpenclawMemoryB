## 📋 概述

金蝶云苍穹是金蝶推出的企业级PaaS平台，支持公有云、私有云和混合云部署。本文档提供详细的部署配置指南，涵盖环境准备、系统部署、配置优化和验证测试等全流程。

### 文档适用范围
- ✅ 私有云生产环境部署
- ✅ 私有云测试环境部署
- ✅ 混合云架构部署
- ❌ 公有云SaaS环境（直接访问 https://dev.kingdee.com）

---

## 🏗️ 部署架构

### 公有云部署
- **适用场景**: 中小企业、快速上线
- **特点**: 无需自建基础设施、按需付费
- **访问方式**: https://dev.kingdee.com

### 私有云部署
- **适用场景**: 大型企业、数据安全要求高
- **特点**: 独立部署、完全掌控
- **要求**: 
  - 服务器资源
  - 网络环境
  - 安全配置

### 混合云部署
- **适用场景**: 业务系统在私有云、部分服务在公有云
- **特点**: 灵活性高、成本优化

#### 2.2 域名和SSL配置

**Nginx配置**:
```nginx
# /etc/nginx/conf.d/k3cloud.conf

# HTTP重定向到HTTPS
server {
    listen 80;
    server_name k3cloud.yourdomain.com;
    
    # LetsEncrypt验证路径
    location /.well-known/acme-challenge/ {
        root /var/www/certbot;
    }
    
    # 其他请求重定向到HTTPS
    location / {
        return 301 https://$server_name$request_uri;
    }
}

# HTTPS配置
server {
    listen 443 ssl http2;
    server_name k3cloud.yourdomain.com;
    
    # SSL证书配置
    ssl_certificate /etc/nginx/ssl/k3cloud.yourdomain.com.crt;
    ssl_certificate_key /etc/nginx/ssl/k3cloud.yourdomain.com.key;
    
    # SSL优化配置
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;
    ssl_prefer_server_ciphers on;
    ssl_session_cache shared:SSL:10m;
    ssl_session_timeout 10m;
    
    # 安全头
    add_header Strict-Transport-Security "max-age=31536000; includeSubDomains" always;
    add_header X-Frame-Options "SAMEORIGIN" always;
    add_header X-Content-Type-Options "nosniff" always;
    add_header X-XSS-Protection "1; mode=block" always;
    
    # 客户端请求大小限制
    client_max_body_size 100M;
    
    # 代理到应用服务器
    location / {
        proxy_pass http://k3cloud_backend;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        # WebSocket支持
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        
        # 超时配置
        proxy_connect_timeout 60s;
        proxy_send_timeout 60s;
        proxy_read_timeout 60s;
    }
    
    # 静态资源缓存
    location ~* \.(jpg|jpeg|png|gif|ico|css|js|pdf|txt|woff|woff2|ttf|eot|svg)$ {
        proxy_pass http://k3cloud_backend;
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
}

# 负载均衡配置
upstream k3cloud_backend {
    least_conn;  # 最少连接算法
    
    server 192.168.1.101:8080 weight=3;
    server 192.168.1.102:8080 weight=3;
    server 192.168.1.103:8080 weight=2;
    
    # 健康检查
    keepalive 32;
}
```

**创建数据库和用户**:
```sql
-- 连接PostgreSQL
psql -U postgres

-- 创建用户

-- 创建数据库
CREATE DATABASE k3cloud_db OWNER k3cloud ENCODING 'UTF8';

-- 连接到k3cloud_db
\c k3cloud_db

-- 授权Schema权限

@Service
public class BillService {

    @Autowired
    private RedissonClient redissonClient;

    public void submitBill(String billId) {
        String lockKey = "bill:lock:" + billId;
        RLock lock = redissonClient.getLock(lockKey);

        try {
            // 尝试获取锁，最多等待5秒，锁定10秒后自动释放
            boolean acquired = lock.tryLock(5, 10, TimeUnit.SECONDS);
            if (acquired) {
                // 执行业务逻辑
                processBill(billId);
            } else {
                throw new BusinessException("获取锁失败，请稍后重试");
            }
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
            throw new BusinessException("操作被中断", e);
        } finally {
            if (lock.isHeldByCurrentThread()) {
                lock.unlock();
            }
        }
    }

# 时间戳
TIMESTAMP=$(date +%Y%m%d_%H%M%S)
BACKUP_FILE="${BACKUP_DIR}/k3cloud_${TIMESTAMP}.sql.gz"

# 创建备份目录
mkdir -p ${BACKUP_DIR}

# 执行备份
echo "开始备份数据库: ${TIMESTAMP}"
pg_dump -h ${DB_HOST} -p ${DB_PORT} -U ${DB_USER} -d ${DB_NAME} \
  --format=plain --no-owner --no-acl | gzip > ${BACKUP_FILE}

# 检查备份是否成功
if [ $? -eq 0 ]; then
    echo "备份成功: ${BACKUP_FILE}"

    # 计算MD5
    md5sum ${BACKUP_FILE} > ${BACKUP_FILE}.md5

    # 上传到远程存储（如OSS）
    # aws s3 cp ${BACKUP_FILE} s3://your-bucket/postgresql/

    # 清理旧备份
    find ${BACKUP_DIR} -name "k3cloud_*.sql.gz" -mtime +${RETENTION_DAYS} -delete
    echo "清理${RETENTION_DAYS}天前的备份完成"
else
    echo "备份失败！"
    # 发送告警
    curl -X POST https://your-webhook-url \
      -d "alert=数据库备份失败&time=${TIMESTAMP}"
    exit 1
fi

Access to XMLHttpRequest at 'http://api.example.com' from origin 'http://frontend.example.com'

**解决方案**:

@Configuration
public class CorsConfig implements WebMvcConfigurer {

    @Override
    public void addCorsMappings(CorsRegistry registry) {
        registry.addMapping("/api/**")
            .allowedOrigins("http://frontend.example.com")
            .allowedMethods("GET", "POST", "PUT", "DELETE", "OPTIONS")
            .allowedHeaders("*")
            .allowCredentials(true)
            .maxAge(3600);
    }
}
```

**Nginx配置**:
```nginx

    if ($request_method = 'OPTIONS') {
        add_header 'Access-Control-Allow-Origin' 'http://frontend.example.com';
        add_header 'Access-Control-Allow-Methods' 'GET, POST, PUT, DELETE, OPTIONS';
        add_header 'Access-Control-Allow-Headers' 'DNT,User-Agent,X-Requested-With,If-Modified-Since,Cache-Control,Content-Type,Range,Authorization';
        add_header 'Access-Control-Max-Age' 1728000;
        add_header 'Content-Type' 'text/plain; charset=utf-8';
        add_header 'Content-Length' 0;
        return 204;
    }

    add_header 'Access-Control-Allow-Origin' 'http://frontend.example.com' always;
    add_header 'Access-Control-Allow-Credentials' 'true' always;

    proxy_pass http://k3cloud_backend;
}
```

#### 问题10：WebSocket连接失败

**现象**:
```
WebSocket connection to 'wss://k3cloud.example.com/ws' failed:
Error during WebSocket handshake: Unexpected response code: 400
```

**解决方案**:

**Nginx配置**:
```nginx
map $http_upgrade $connection_upgrade {
    default upgrade;
    '' close;
}

upstream websocket {
    server 192.168.1.101:8080;
}

server {
    listen 443 ssl;
    server_name k3cloud.example.com;

    location /ws {
        proxy_pass http://websocket;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection $connection_upgrade;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_read_timeout 86400;
    }
}
```

**Spring Boot配置**:
```java
@Configuration
@EnableWebSocket
public class WebSocketConfig implements WebSocketConfigurer {

    @Override
    public void registerWebSocketHandlers(WebSocketHandlerRegistry registry) {
        registry.addHandler(new K3CloudWebSocketHandler(), "/ws")
            .setAllowedOrigins("https://k3cloud.example.com")
            .addInterceptors(new WebSocketHandshakeInterceptor());
    }
}
```
