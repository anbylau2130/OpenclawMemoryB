# 用户场景

随着企业数字化进入深水区，越来越多的业务性场景需求被发掘出来。这类需求通常紧急、零散、数量多、难复用、变更快，需要快速上线进行验证、试错并应用。然而，以IT部门为核心的开发模式，往往优先建设企业的核心业务系统，追求业务的标准化与可靠性。面对海量的个性化业务需求，IT部门需要付出高昂的沟通成本、开发成本和验证成本，使得他们无法快速响应业务部门，为其提供精细化、个性化的IT建设，最终导致业务难以实现创新与个性化执行，从而增加了企业的管理成本。为了更好地解决以上问题，让创新力进一步下沉到业务部门，让IT建设深入到企业的每一个角落，苍穹无代码开发平台，来了！

苍穹无代码开发平台是基于苍穹低代码开发平台的先进的模型驱动为底座，帮助企业客户推动创造力下沉，缩短业务和数字化的距离，业务负责人和基层员工直接开展IT建设，让管理想法第一时间落地，减轻IT部门的压力等数字化工作的开展。产品追求的核心体验是易上手、低门槛，让业务人员使用业务语言即可完成全部表单、流程、逻辑自动化、统计等配置；支撑创新、快速试错的新型业务的数字化、小单元组织对相同业务执行差异化管理的数字化等业务场景

支撑的业务主要具备特征包含：小团队、基层业务、创新型业务、实验性流程、频繁调整、差异化执行、使用频率高、日常办公、非正式协作
典型场景如：敏捷协作、调研问卷&投票、办公用品领用、会议室管理、停车管理

# 功能介绍

无代码开发平台提供以下主要功能：

1. **应用**：允许用户创建和使用应用程序，基于应用内的表单、流程、分析来满足个性化场景对数字化工具的需求
2. **表单**：让用户能够根据业务需求创建数据收集或信息提交的表单，能够快速低成本地满足特定的业务数据或信息收集需求
3. **流程**：允许用户通过拖拽和配置各类型节点来创建和设计业务流程，提高业务流程的自动化程度和标准化程度，提高工作效率
4. **分析**：提供了一种快速分析数据的工具，使用户能够通过可视化图表、数据透视表等工具对表单和流程中的数据进行深入的分析和探索，可以提高数据分析的准确性和效率，更好地理解和利用数据
5. **模板**：允许用户通过模板来快速创建、使用、分享应用、表单、流程，可以提高创建效率，减少设计工作量
6. **工作台**：允许用户在工作台中配置自定义卡片，专注于常用、习惯的操作，可以提高用户操作便捷性，个性化配置操作路径
7. **平台管理**：允许管理员对平台进行全局配置和管理，可以提高平台的稳定性和可维护性，确保平台的正常运行和安全管理
8. **移动端**：提供了无代码开发平台的移动端使用界面，可以满足用户在移动场景下使用无代码开发平台的应用、表单、流程、分析、工作台等功能
