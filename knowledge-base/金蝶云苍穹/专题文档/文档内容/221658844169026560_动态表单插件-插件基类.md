## 1 插件基类介绍

动态表单插件的基类为 **AbstractFormPlugin**。动态表单插件，必须从插件基类AbstractFormPlugin中派生。

## 2 插件基类定义

```java
package kd.bos.form.plugin;

public abstract class AbstractFormPlugin 
    implements IFormPlugin, 
           IPlugin, 
           IDataModelListener, 
           IControlPlugin {
```

## 3 主要功能

插件基类AbstractFormPlugin已经实现了动态表单插件的大部分事件接口，包括：

- initialize - 插件初始化
- registerListener - 注册监听器
- afterBindData - 数据绑定后
- click - 按钮点击
- propertyChanged - 属性变更
- closedCallBack - 关闭回调
- confirmCallBack - 确认回调
- beforeDoOperation - 操作执行前
- afterDoOperation - 操作执行后

## 4 核心方法

| 方法 | 说明 |
|-----|------|
| getView() | 获取表单视图模型 |
| getModel() | 获取表单数据模型 |
| getPageId() | 获取页面ID |
| getControl(String key) | 获取控件实例 |

## 5 使用示例

```java
package kd.bos.plugin.sample;

import kd.bos.form.plugin.AbstractFormPlugin;
import java.util.EventObject;

public class MyFormPlugin extends AbstractFormPlugin {
    
    @Override
    public void afterBindData(EventObject e) {
        super.afterBindData(e);
        // 数据绑定后的处理逻辑
    }
    
    @Override
    public void click(EventObject evt) {
        super.click(evt);
        // 按钮点击处理逻辑
    }
}
```
