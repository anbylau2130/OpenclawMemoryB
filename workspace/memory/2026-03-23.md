## Ralph Loop 方法论

### 核心概念
Ralph Loop 是一个自主 AI 编程循环，通过不断启动新的 AI 实例来完成任务，直到所有 PRD 项完成。

**最简单的形式：**
```bash
while :; do cat PROMPT.md | claude-code; done
```

### 核心原则
1. **单任务循环（One Item Per Loop）** - 每次循环只做一个任务
2. **三阶段流程** - Generate → Backpressure → Commit
3. **记忆持久化** - prd.json + progress.txt + specs/

### Skill 文件结构
```
skills/ralph-loop/
├── SKILL.md                    # 完整方法论文档（6KB）
├── README.md                   # 快速开始指南
├── scripts/ralph-loop.sh      # 自动化脚本
├── templates/
│   ├── prd.json               # PRD 模板
│   ├── progress.txt           # 进度文件模板
│   ├── prompt.md              # Prompt 模板
│   ├── specs/                 # 示例规格文档
│   └── stdlib/                # 代码模式标准库
```

### 适用场景
- ✅ Greenfield 项目
- ✅ 有明确规格的任务
- ✅ 有完善的测试体系
- ✅ 单一仓库

### 参考资料
- Geoffrey Huntley 博客：https://ghuntley.com/ralph/
- Ralph GitHub：https://github.com/snarktank/ralph

## Codegen-LSP 代码生成器

### 学习 lsprotocol 方法论

**来源：** https://github.com/microsoft/lsprotocol

**核心概念：**
- **单一数据源** - 一个 JSON 规范，多种语言实现
- **插件架构** - 核心固定，插件扩展
- **模板驱动** - 数据与表现分离
- **自动化** - 工具链集成

**设计模式：**
1. 策略模式 - 不同语言使用不同生成策略
2. 模板方法模式 - 生成流程固定，细节由子类实现
3. 工厂模式 - 根据语言名创建插件

**支持的语言：**
- Python（dataclass/pydantic）
- Rust（struct/enum）
- Dotnet（class/record）
- Crystal、Golang

### Codegen-LSP Skill 创建

**位置：** `skills/codegen-lsp/`

**文件结构：**
```
codegen-lsp/
├── SKILL.md                    # 完整方法论文档（10KB）
├── README.md                   # 快速开始指南（3.6KB）
├── scripts/
│   └── generate.py            # 主生成器（11KB）
├── templates/
│   ├── python/
│   │   ├── class.jinja2
│   │   └── init.jinja2
│   ├── dotnet/
│   │   └── class.jinja2
│   └── sqlserver/
│       ├── table.jinja2
│       └── procedure.jinja2
└── examples/
    └── ecommerce.json         # 电商示例
```

**核心特性：**
- 规范驱动（JSON 格式）
- 类型安全（类型映射）
- 模板引擎（Jinja2）
- 自动生成（CRUD + 索引）

**支持的语言：**
- **Python** - dataclass、类型提示、__init__.py
- **Dotnet (C#)** - Entity Framework、Data Annotations
- **SQL Server** - CREATE TABLE、存储过程、索引

**类型映射：**

| 规范类型 | Python | Dotnet | SQL Server |
|---------|--------|--------|-----------|
| `integer` | int | int | INT |
| `string` | str | string | NVARCHAR |
| `decimal` | Decimal | decimal | DECIMAL |
| `datetime` | datetime | DateTime | DATETIME2 |
| `boolean` | bool | bool | BIT |
| `guid` | UUID | Guid | UNIQUEIDENTIFIER |
| `binary` | bytes | byte[] | VARBINARY |

**使用方法：**
```bash
# 生成所有语言
python scripts/generate.py --input examples/ecommerce.json --output ./generated

# 只生成 Python
python scripts/generate.py -i examples/ecommerce.json -l python -o ./models

# 预览生成内容
python scripts/generate.py -i examples/ecommerce.json -l python --dry-run
```

**高级功能：**
- 枚举支持
- 外键关系
- 自定义表名
- 索引定义
- 自定义模板

**与 Ralph Loop 集成：**
- 可作为 Ralph Loop 的子任务使用
- 自动生成数据模型代码
- 规范更新后自动同步
