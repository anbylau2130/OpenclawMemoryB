## 容器自动启动配置完成

**时间：** 15:28

### 配置内容

修改 `/etc/init.d/pm2-undefined`，添加 watchdog 自动启动：

```bash
start() {
    echo "Starting $NAME"
    super $PM2 resurrect
    # 启动 watchdog 监控
    if [ -f /root/.openclaw/workspace/scripts/pm2-watchdog.sh ]; then
        su - root -c "nohup /root/.openclaw/workspace/scripts/pm2-watchdog.sh > /root/.openclaw/workspace/logs/pm2-watchdog.out 2>&1 &"
        echo "Started PM2 watchdog"
    fi
}
```

### 自动启动流程

```
容器启动 → init.d → PM2 resurrect → Dashboard 恢复 → Watchdog 启动
```

## 太子配置错误修复

**时间：** 15:40-15:43

### 问题
Gateway 启动失败，报错：
```
Invalid config: agents.list.2.subagents: Unrecognized key: "enabled"
```

### 原因
尝试添加 `subagents.enabled = true` 以启用太子派发功能，但 OpenClaw 不支持此字段。

### 解决方案
移除所有 agent 的 `subagents.enabled` 配置：
```python
for agent in config['agents']['list']:
    if 'subagents' in agent and 'enabled' in agent['subagents']:
        del agent['subagents']['enabled']
```
