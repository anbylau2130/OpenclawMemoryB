# 对话记录系统

## 📋 功能说明

**自动记录用户说的话**，确保对话历史的完整性和可追溯性。

---

## 🎯 核心功能

### 1. 自动记录
- ✅ 记录用户发送的每条消息
- ✅ 按日期保存（YYYY-MM-DD.md）
- ✅ 只记录私聊对话
- ✅ 只保存用户说的内容

### 2. 记录格式
```markdown
# 2026-03-24 对话记录

**记录规则:** 只记录私聊对话，只保存用户说的内容

---

## 07:35
- "中书省外做什么"

## 08:14
- "但是我为什么在gui中看不到他们在做什么？"

---

_最后更新: 2026-03-24 18:27_
```

---

## 🔧 使用方法

### 自动记录（推荐）
系统会在收到用户消息时自动调用记录器。

### 手动记录
```python
from conversation_recorder import ConversationRecorder

recorder = ConversationRecorder()
recorder.record_user_message("这是一条测试消息")
```

### 查看统计
```python
stats = recorder.get_stats()
# {"date": "2026-03-24", "total_conversations": 31}
```

### 获取今日对话
```python
conversations = recorder.get_today_conversations()
# [{"time": "07:35", "message": "中书省外做什么"}]
```

## 🎯 记录规则

1. **只记录私聊** - 不记录群聊对话
2. **只保存用户说的内容** - 不保存agent回复
3. **按日期保存** - 每天一个文件
4. **自动更新时间戳** - 每次记录后更新

## ⚠️ 重要说明

### 为什么需要这个系统？
之前的系统只保存agent回复，**不保存用户说的话**，导致：
- ❌ 无法完整追溯对话
- ❌ 无法复盘用户意图
- ❌ 无法优化对话体验

### 解决方案
- ✅ 自动记录每条用户消息
- ✅ 完整保存对话历史
- ✅ 支持历史对话检索
