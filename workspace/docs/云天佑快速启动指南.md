## 🚀 快速启动（3种方式）

### 方式1: Docker Compose（推荐）⭐
```bash

docker-compose up -d
```

**访问地址**:
- **前端**: http://localhost:3000
- **后端API**: http://localhost:5000
- **Swagger文档**: http://localhost:5000/swagger
- **默认账号**: admin@example.com / Admin123!

**优点**: 一键启动，自动配置， 开箱即用
