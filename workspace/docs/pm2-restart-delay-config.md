## 📦 配置详情

### PM2启动参数
```bash
pm2 start index.js \
  --name boluo-dashboard \
  --restart-delay=3000
```

### 配置参数
- **restart-delay**: 3000ms（3秒）
- **说明**: Dashboard崩溃后等待3秒再重启，给端口释放时间
