## 📄 文件命名规则

**格式：** `{股票名称}/{股票名称}-{日期}-{扫描次数}.md`

**示例：**
```
三一重工/三一重工-2026-03-20-30.md
```

**说明：**
- 股票名称：中文股票名称
- 日期：YYYY-MM-DD格式
- 扫描次数：监控扫描次数（递增）

## 📊 报告内容

每份报告包含：

### 1. 基础数据
- 当前价格
- 涨跌幅
- 因子得分
- RSI
- KDJ-K
- VWAP
- 布林下轨
- 信号

### 2. 量价分析
- 量价形态（9种）
- 量比
- 平均成交量
- 当前成交量
- 市场含义
- 交易建议
- 信号强度

### 3. 市场情绪
- 情绪得分（0-100）
- 情绪等级（5级）
- 情绪描述
- 情绪建议
- 分项得分（5维度）

### 4. 交易建议
- 短期策略（1-2周）
- 中期策略（1个月）
- 长期策略（3个月）

### 5. 优化结果
- 综合信心度
- 风险等级
- 最终建议

### 6. 风险提示
- 止损纪律
- 仓位管理
- 禁止操作

## 🔧 使用方法

### 查看某只股票的所有报告
```bash
ls -lh /root/.openclaw/workspace/trading/三一重工/
```

### 查看最新报告
```bash
ls -lt /root/.openclaw/workspace/trading/三一重工/ | head -1
cat /root/.openclaw/workspace/trading/三一重工/三一重工-2026-03-20-31.md
```

### 查看某日所有股票报告
```bash
find /root/.openclaw/workspace/trading -name "*2026-03-20*.md"
```

### 统计某只股票报告数量
```bash
ls /root/.openclaw/workspace/trading/三一重工/*.md | wc -l
```

## 🚀 自动生成

**生成频率：** 每30分钟

**生成时间：** 交易时段（9:30-15:00）

**监控股票：** 10只上证50成分股

**通知机制：**
- 得分 < 3.5：只生成报告，不通知
- 得分 ≥ 3.5：生成报告 + 钉钉通知
