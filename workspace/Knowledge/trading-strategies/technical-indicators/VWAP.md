## 📚 一、VWAP 基础概念

### 1.1 什么是 VWAP

**定义：**
- Volume Weighted Average Price（成交量加权平均价格）
- 按成交量加权的平均价格
- 反映真实交易成本

**公式：**

```
VWAP = Σ(价格 × 成交量) / Σ(成交量)

其中：
- 价格：通常是典型价格 = (最高价 + 最低价 + 收盘价) / 3
- 成交量：对应价格的成交量
```

**示例计算：**

```
时间段  最高价  最低价  收盘价  成交量   典型价格   价格×成交量
10:00   22.50   22.30   22.40   1000     22.40      22,400
10:05   22.45   22.35   22.40   1500     22.40      33,600
10:10   22.60   22.40   22.50   2000     22.50      45,000
10:15   22.55   22.45   22.50   1200     22.50      27,000

VWAP = (22,400 + 33,600 + 45,000 + 27,000) / (1000 + 1500 + 2000 + 1200)
     = 128,000 / 5,700
     = 22.46元
```

---

### 1.2 VWAP 的特点

| 特点 | 说明 |
|------|------|
| **动态性** | 随时间推移不断更新 |
| **成交量加权** | 反映真实交易成本 |
| **支撑阻力** | 价格常在VWAP附近波动 |
| **机构基准** | 机构常用VWAP评估交易执行 |

**优势：**
- ✅ 反映真实市场成本
- ✅ 动态调整，适应市场
- ✅ 支撑阻力作用明显
- ✅ 机构广泛使用

**劣势：**
- ❌ 只适用于日内交易
- ❌ 需要分时数据
- ❌ 对低流动性股票效果差

## 📈 二、VWAP 交易信号

### 2.1 基本信号

**买入信号：**

```
1. 价格上穿VWAP
   - 价格从下方穿越VWAP
   - 成交量放大
   - 趋势转强

2. 价格回踩VWAP获得支撑
   - 价格上涨后回调至VWAP
   - 未跌破VWAP
   - 成交量萎缩
   - 再次上涨

3. VWAP向上倾斜
   - VWAP曲线向上
   - 市场买盘强劲
```

**卖出信号：**

```
1. 价格下穿VWAP
   - 价格从上方穿越VWAP
   - 成交量放大
   - 趋势转弱

2. 价格反弹至VWAP遇阻
   - 价格下跌后反弹至VWAP
   - 未突破VWAP
   - 成交量萎缩
   - 再次下跌

3. VWAP向下倾斜
   - VWAP曲线向下
   - 市场卖盘强劲
```

## 🎯 三、VWAP 实战应用

### 3.1 日内交易流程

**开盘前（9:15-9:30）：**
```
1. 查看隔夜消息
2. 检查大盘环境
3. 确定观察标的
4. 设置价格提醒
```

**开盘观察（9:30-10:00）：**
```
1. 观察开盘价与VWAP关系
   - 高开 > VWAP：强势
   - 低开 < VWAP：弱势

2. 观察成交量
   - 放量：活跃
   - 缩量：谨慎

3. 等待第一根30分钟K线
   - 确定日内方向
```

**交易时段（10:00-11:30, 13:00-14:30）：**
```
1. 等待VWAP信号
   - 价格上穿VWAP
   - 回踩确认
   - 放量启动

2. 执行交易
   - 买入/卖出
   - 设置止损止盈

3. 持仓管理
   - 监控VWAP位置
   - 调整止损止盈
```

**尾盘处理（14:30-15:00）：**
```
1. 日内交易：平仓离场
2. 波段交易：持仓过夜（符合条件）
3. 记录交易日志
```

## 💡 五、VWAP 常见陷阱

### 5.1 强趋势中的假信号

**问题：**

```
强上涨趋势：
- 价格长期在VWAP上方
- 回踩VWAP可能只是短暂调整
- 卖出信号失效

强下跌趋势：
- 价格长期在VWAP下方
- 反弹VWAP可能只是短暂反弹
- 买入信号失效
```

**解决方案：**

```
1. 结合趋势判断
   - MA20向上 → 忽略VWAP卖出信号
   - MA20向下 → 忽略VWAP买入信号

2. 等待确认信号
   - 突破+回踩+放量
   - 多重确认

3. 调整止损止盈
   - 强趋势：扩大止损止盈
   - 震荡市：缩小止损止盈
```

---

### 5.2 低流动性股票

**问题：**

```
低流动性股票：
- 成交量小
- VWAP不稳定
- 容易被操纵
- 信号失效
```

**解决方案：**

```
1. 筛选流动性好的股票
   - 日成交额 > 1亿
   - 换手率 > 1%

2. 避开低流动性时段
   - 午休时间（11:30-13:00）
   - 尾盘最后10分钟

3. 结合成交量确认
   - 信号出现时成交量放大
   - 避免成交量萎缩时的信号
```

---

### 5.3 开盘和尾盘

**问题：**

```
开盘（9:30-10:00）：
- VWAP刚开始计算
- 数据不充分
- 信号不可靠

尾盘（14:30-15:00）：
- 成交量异常
- 价格波动大
- 信号失真
```

**解决方案：**

```
1. 避开开盘前30分钟
   - 等待VWAP稳定
   - 10:00后开始交易

2. 避开尾盘最后30分钟
   - 14:30前完成交易
   - 或持仓过夜

3. 特殊情况
   - 开盘跳空：等待回补
   - 尾盘拉升：谨慎追高
```

### 6.3 风险控制

**单笔风险：**

```
止损：VWAP - 2-3%
计算：止损位 = VWAP × (1 - 2-3%)

示例：
VWAP：22.50元
止损位：22.50 × 0.97 = 21.83元
最大亏损：-3%
```

**总风险：**

```
总仓位：≤ 80%
单笔仓位：≤ 60%
保留现金：≥ 20%

总风险计算：
总风险 = 单笔风险 × 仓位比例
示例：3% × 60% = 1.8%（安全）
```
