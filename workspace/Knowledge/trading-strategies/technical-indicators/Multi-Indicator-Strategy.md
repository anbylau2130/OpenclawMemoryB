# 技术指标综合应用 - 多指标组合策略

## 一、多指标组合原理

### 1.1 为什么要组合使用

**单一指标的局限：**

```
1. 假信号多
   - RSI 超买但价格继续上涨
   - KDJ 金叉但价格下跌
   - MACD 背离失败

2. 滞后性
   - MACD 信号滞后
   - 等确认时已错过最佳点位

3. 适用性有限
   - RSI 适合震荡市
   - MACD 适合趋势市
   - 单一指标无法适应所有市场
```

**多指标组合的优势：**

```
1. 提高胜率
   - 单一指标胜率：65-70%
   - 双重确认胜率：75-80%
   - 三重确认胜率：85-90%
   - 四重确认胜率：90-95%

2. 降低假信号
   - 多重过滤
   - 相互验证

3. 适应性强
   - 不同市场环境
   - 不同交易风格
```

---

### 1.2 指标分类

**按类型分类：**

| 类型 | 指标 | 特点 | 适用场景 |
|------|------|------|---------|
| **趋势指标** | MACD、MA | 滞后，稳定 | 趋势跟踪 |
| **震荡指标** | RSI、KDJ | 敏感，波动大 | 震荡操作 |
| **波动性指标** | BOLL | 动态轨道 | 波动分析 |
| **量价指标** | 成交量、VWAP | 最直接 | 量价分析 |

**按速度分类：**

| 速度 | 指标 | 特点 | 适用场景 |
|------|------|------|---------|
| **快速** | KDJ、RSI(6) | 敏感，信号多 | 短线交易 |
| **中速** | RSI(14)、BOLL | 适中，稳定 | 波段交易 |
| **慢速** | MACD、MA | 稳定，信号少 | 趋势跟踪 |

---

### 1.3 组合原则

**原则1：不同类型组合**

```
✅ 好的组合：
- 趋势指标 + 震荡指标（MACD + RSI）
- 震荡指标 + 波动性指标（KDJ + BOLL）
- 量价 + 技术指标（成交量 + MACD）

❌ 不好的组合：
- 同类型指标（RSI + KDJ，都是震荡指标）
- 功能重复（RSI(14) + RSI(6)，只是周期不同）
```

**原则2：速度互补**

```
✅ 好的组合：
- 快速 + 中速（KDJ + RSI）
- 中速 + 慢速（RSI + MACD）
- 快速 + 慢速（KDJ + MACD）

❌ 不好的组合：
- 同速度指标（KDJ + RSI(6)，都很快）
```

**原则3：信号确认**

```
买入信号：
1. MACD 金叉（趋势确认）
2. RSI 底背离（反转信号）
3. KDJ 金叉（短线确认）
4. BOLL 触及下轨（支撑确认）
5. 缩量（量价确认）

确认越多，胜率越高
```

## 二、经典组合策略

### 2.1 MACD + RSI 组合

**策略1：MACD趋势 + RSI超卖**

```
买入信号：
1. MACD 零轴下金叉（趋势反转）
2. RSI < 30（超卖）
3. 成交量萎缩（量价确认）

胜率：80-85%

示例：三一重工
2026-03-04
MACD：零轴下金叉
RSI：28（超卖）
成交量：缩量43%
判断：MACD趋势反转 + RSI超卖 = 买入信号
```

**策略2：MACD背离 + RSI背离**

```
卖出信号：
1. MACD 顶背离（价格新高，DIF未新高）
2. RSI 顶背离（价格新高，RSI未新高）
3. 成交量放大（量价确认）

胜率：85-90%

示例：
价格：22.70元（新高）
MACD：DIF=0.08（未新高，前高0.12）
RSI：72（未新高，前高78）
成交量：放大
判断：双重顶背离 = 强烈卖出信号
```

## 三、交易系统构建

### 3.1 完整交易系统

**系统架构：**

```
1. 标的筛选模块
   - 成交额 > 1亿
   - MA20 方向明确
   - 有锚定标的

2. 信号确认模块
   - 量价信号
   - MACD 信号
   - KDJ 信号
   - RSI 信号
   - BOLL 信号

3. 信号强度模块
   - 单一信号：弱（不操作）
   - 双重信号：中（观察）
   - 三重信号：强（准备操作）
   - 四重信号：极强（立即操作）
   - 五重信号：极强（满仓操作）

4. 仓位管理模块
   - 弱信号：0% 仓位
   - 中信号：20% 仓位
   - 强信号：50% 仓位
   - 极强信号：80% 仓位
   - 保留20%机动资金

5. 风险控制模块
   - 单笔止损：-7%
   - 总止损：-20%
   - 信号失败：立即止损
   - 背离失败：立即止损

6. 执行模块
   - 开盘观察（9:30-10:00）
   - 信号确认（10:00-10:30）
   - 执行交易（10:30-11:00）
   - 记录日志（盘后）
```

## 四、风险控制

### 4.1 止损规则

**技术止损：**

```
1. 信号失败止损
   - 买入后信号消失
   - 立即止损

2. 背离失败止损
   - 背离后价格继续下跌
   - 立即止损

3. 跌破支撑位止损
   - 跌破支撑位 + 放量
   - 立即止损

4. 指标反转止损
   - MACD死叉
   - KDJ死叉
   - 立即止损
```

**资金止损：**

```
1. 单笔止损
   - 最大亏损：-7%
   - 计算公式：止损位 = 买入价 × (1 - 7%)
   - 机械执行，不犹豫

2. 总止损
   - 最大总亏损：-20%
   - 计算公式：总亏损 = Σ(单笔亏损 × 仓位比例)
   - 触及时停止交易1周

3. 连续止损
   - 连续3次止损
   - 停止交易3天
   - 复盘总结
```

---

### 4.2 止盈规则

**分批止盈：**

```
1. 第1次止盈（触及阻力1）
   - 卖出30%
   - 锁定部分利润

2. 第2次止盈（触及阻力2）
   - 卖出30%
   - 继续锁定利润

3. 第3次止盈（触及阻力3）
   - 卖出40%
   - 清仓或保留底仓
```

**移动止盈：**

```
1. 最高价回撤5%止盈
   - 适用于强势股
   - 保护利润

2. MACD柱缩小止盈
   - 多头动能减弱
   - 准备离场

3. 顶背离确认止盈
   - 趋势反转信号
   - 立即清仓
```

---

### 4.3 信号失败处理

**失败类型：**

```
1. 假突破
   - 突破后立即回落
   - 止损离场

2. 背离失败
   - 背离后价格继续原方向
   - 止损离场

3. 金叉死叉失败
   - 金叉后价格下跌
   - 死叉后价格上涨
   - 止损离场

4. 超买超卖失败
   - 超买后继续上涨
   - 超卖后继续下跌
   - 等待反转或止损
```

**处理流程：**

```
1. 立即止损
   - 不抱有幻想
   - 机械执行

2. 记录失败原因
   - 市场环境变化
   - 指标失效
   - 操作失误

3. 复盘总结
   - 分析失败原因
   - 优化策略
   - 避免重复犯错

4. 暂停交易
   - 连续失败3次
   - 停止交易1-3天
   - 冷静反思
```
