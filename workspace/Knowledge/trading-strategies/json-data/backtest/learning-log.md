#### 2. 信号识别逻辑

**价涨量增识别：**
```
条件1: 收盘价 > 前日收盘价
条件2: 成交量 > 前日成交量
条件3: 成交量 > MA5成交量
强度: 放量倍数决定（1-1.5倍=弱，1.5-2倍=中，>2倍=强）
```

**价跌量缩识别：**
```
条件1: 收盘价 < 前日收盘价
条件2: 成交量 < 前日成交量
条件3: 连续3天以上缩量下跌 = 卖盘枯竭
```

**支撑位计算：**
```
1. MA20: 最近20日均价
2. 前低: 最近60日最低价
3. 黄金分割: 高点 - (高点-低点) × 0.382/0.5/0.618
```
