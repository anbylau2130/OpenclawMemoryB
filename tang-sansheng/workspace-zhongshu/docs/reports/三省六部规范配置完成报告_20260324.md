## 🔍 Message 工具说明

### 为什么其他部门没有回复？

**Message 工具是单向通知，不会触发 agent 自动回复！**

```
中书省 → message → 其他部门
  ↓
✅ 消息已送达（messageId 确认）
❌ 但不会触发自动回复
```

### Message 工具的正确用途

1. ✅ **发送通知** - 单向传递信息
2. ✅ **广播消息** - 批量通知
3. ❌ **不适合** - 触发对话或获取回复

### 如何获取其他部门回复？

**方案1**: 陛下直接对话
- 打开钉钉
- 找到对应部门机器人
- 发送消息触发对话

**方案2**: 通过尚书省spawn
- 尚书省有spawn权限
- 可以spawn其他agent执行任务

**方案3**: 配置到AGENTS.md（已完成）
- Agent启动时会看到规范
- 下次对话时自动遵守
