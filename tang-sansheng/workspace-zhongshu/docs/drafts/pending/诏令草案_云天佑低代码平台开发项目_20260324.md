## 💻 技术栈

### 后端
- **语言**: C# .NET Core 8
- **框架**: ASP.NET Core 8
- **数据库**: PostgreSQL 15+
- **缓存**: Redis 7+
- **消息队列**: RabbitMQ
- **ORM**: Entity Framework Core 8
- **认证**: JWT Bearer
- **日志**: Serilog
- **API文档**: Swagger/OpenAPI

### 前端
- **框架**: React 18+
- **语言**: TypeScript 5+
- **UI库**: Ant Design 5+
- **状态管理**: Redux Toolkit
- **路由**: React Router 6+
- **HTTP客户端**: Axios
- **图表**: ECharts 5+
- **表单**: React Hook Form
- **拖拽**: React DnD / dnd-kit

### DevOps
- **容器**: Docker
- **编排**: Kubernetes
- **CI/CD**: GitLab CI / GitHub Actions
- **监控**: Prometheus + Grafana
- **日志**: ELK Stack
