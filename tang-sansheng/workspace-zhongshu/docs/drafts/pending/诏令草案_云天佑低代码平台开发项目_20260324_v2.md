## 📋 项目概述

### 项目目标
根据金蝶云苍穹76个文档，开发**云天佑低代码平台**，功能与云苍穹**完全一致**。

### 核心要求
1. ✅ **功能完全一致** - 云苍穹有的功能，云天佑也要有
2. ✅ **技术栈升级** - 使用最新的 .NET Core 10
3. ✅ **数据库一致** - PostgreSQL 15+（与云苍穹一致）
4. ✅ **多平台应用** - 使用 Blazor 开发前端

---

## 💻 技术栈设计

### 后端技术栈

**核心框架**:
- **.NET Core 10** - 最新版本
- **ASP.NET Core WebAPI** - RESTful API服务
- **Entity Framework Core 10** - ORM框架
- **MediatR** - CQRS模式
- **AutoMapper** - 对象映射
- **FluentValidation** - 数据验证

**数据库**（与云苍穹一致）:
- **PostgreSQL 15+** - 主数据库
- **Redis 7+** - 缓存
- **RabbitMQ** - 消息队列
- **Elasticsearch** - 搜索引擎（可选）

**安全与认证**:
- **JWT Bearer** - 认证
- **IdentityServer** - 身份认证（可选）
- **Serilog** - 日志

**API文档**:
- **Swagger/OpenAPI** - API文档自动生成

---

### 前端技术栈

**核心框架**:
- **Blazor WebAssembly** - Web端
- **Blazor Hybrid (MAUI)** - 桌面端（Windows/macOS/Linux）
- **Blazor Mobile** - 移动端（iOS/Android）

**UI组件库**:
- **MudBlazor** - Material Design风格
- **Ant Design Blazor** - 企业级UI组件

**状态管理**:
- **Fluxor** - 类Redux状态管理
- **Blazor State** - 组件状态管理

**其他**:
- **Blazor IndexedDB** - 本地存储
- **Blazor SignalR** - 实时通信
