## 📋 项目概述

### 项目名称
**Tianyou** - 企业级低代码开发平台

### 项目目标
开发与某云苍穹低代码平台功能**完全一致**的企业级低代码平台，提供**更人性化**的设计。

### 核心要求
1. ✅ 功能完全一致 - 云苍穹有的功能，Tianyou也要有
2. ✅ 更人性化设计 - 优化用户体验
3. ✅ 技术栈先进 - .NET 8 + Blazor（便于维护）
4. ✅ 隐私保护 - 不出现任何第三方厂商名称
5. ✅ **分阶段交付** - MVP核心功能优先

## 💻 技术栈（最终确定）

### 后端
- **.NET 8 LTS** - 长期支持版本（稳定）
- **PostgreSQL 15+** - 主数据库
- **Redis 7+** - 缓存
- **RabbitMQ 3.12+** - 消息队列
- **Entity Framework Core 8** - ORM
- **MediatR** - CQRS模式
- **Serilog** - 日志
- **Swagger/OpenAPI** - API文档

### 前端
- **Blazor WebAssembly** - Web端
- **Blazor MAUI** - 桌面端（Phase 3）
- **Blazor Hybrid** - 移动端（Phase 3）
- **MudBlazor** - Material Design组件库
- **Blazor.SignalR** - 实时通信

### DevOps
- **Docker** - 容器化
- **Kubernetes** - 容器编排
- **GitHub Actions** - CI/CD
- **Prometheus + Grafana** - 监控
- **ELK Stack** - 日志收集

### 技术栈选择理由（陛下确认）

**为什么选择Blazor**：
1. ✅ **统一技术栈** - C#全栈开发，前后端类型共享
2. ✅ **便于维护** - 一套代码兼容所有平台（Web/桌面/移动）
3. ✅ **降低维护成本** - 不需要维护多套技术栈
4. ✅ **代码复用** - 前后端可共享业务逻辑

**风险应对方案**：
1. **Blazor生态不成熟** → 使用成熟组件库（MudBlazor）
2. **性能问题** → 提前性能测试，优化关键路径
3. **多平台适配** → 分阶段实施，先Web后桌面移动
4. **降级方案** → 如多平台失败，优先保证Web端

### 1. 用户与权限

```sql
-- 用户表
CREATE TABLE users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    username VARCHAR(50) UNIQUE NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    full_name VARCHAR(100),
    avatar_url VARCHAR(500),
    status VARCHAR(20) DEFAULT 'active', -- active, inactive, locked
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    last_login_at TIMESTAMP
);

-- 角色表
CREATE TABLE roles (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    role_name VARCHAR(50) UNIQUE NOT NULL,
    role_code VARCHAR(50) UNIQUE NOT NULL,
    description TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 用户角色关联表
CREATE TABLE user_roles (
    user_id UUID REFERENCES users(id) ON DELETE CASCADE,
    role_id UUID REFERENCES roles(id) ON DELETE CASCADE,
    PRIMARY KEY (user_id, role_id)
);

-- 权限表
CREATE TABLE permissions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    permission_name VARCHAR(100) UNIQUE NOT NULL,
    permission_code VARCHAR(100) UNIQUE NOT NULL,
    resource VARCHAR(100), -- 资源类型：form, entity, workflow等
    action VARCHAR(50), -- 操作：create, read, update, delete
    description TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 角色权限关联表
CREATE TABLE role_permissions (
    role_id UUID REFERENCES roles(id) ON DELETE CASCADE,
    permission_id UUID REFERENCES permissions(id) ON DELETE CASCADE,
    PRIMARY KEY (role_id, permission_id)
);

-- 租户表（多租户）
CREATE TABLE tenants (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    tenant_name VARCHAR(100) UNIQUE NOT NULL,
    tenant_code VARCHAR(50) UNIQUE NOT NULL,
    schema_name VARCHAR(50) UNIQUE NOT NULL, -- PostgreSQL Schema隔离
    status VARCHAR(20) DEFAULT 'active',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 用户租户关联表
CREATE TABLE user_tenants (
    user_id UUID REFERENCES users(id) ON DELETE CASCADE,
    tenant_id UUID REFERENCES tenants(id) ON DELETE CASCADE,
    PRIMARY KEY (user_id, tenant_id)
);
```

### 2. 数据模型

```sql
-- 实体模型表
CREATE TABLE entity_models (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    entity_name VARCHAR(100) NOT NULL,
    entity_code VARCHAR(50) UNIQUE NOT NULL,
    table_name VARCHAR(100) NOT NULL,
    description TEXT,
    is_system BOOLEAN DEFAULT FALSE, -- 是否系统实体
    tenant_id UUID REFERENCES tenants(id),
    created_by UUID REFERENCES users(id),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 字段定义表
CREATE TABLE field_definitions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    entity_id UUID REFERENCES entity_models(id) ON DELETE CASCADE,
    field_name VARCHAR(100) NOT NULL,
    field_code VARCHAR(50) NOT NULL,
    field_type VARCHAR(50) NOT NULL, -- text, number, date, select等
    is_required BOOLEAN DEFAULT FALSE,
    is_unique BOOLEAN DEFAULT FALSE,
    default_value TEXT,
    validation_rules JSONB, -- 验证规则（JSON格式）
    properties JSONB, -- 字段属性（JSON格式）
    sort_order INTEGER DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(entity_id, field_code)
);

-- 索引定义表
CREATE TABLE index_definitions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    entity_id UUID REFERENCES entity_models(id) ON DELETE CASCADE,
    index_name VARCHAR(100) NOT NULL,
    index_type VARCHAR(20) DEFAULT 'btree', -- btree, hash, gin等
    fields JSONB NOT NULL, -- 字段列表（JSON格式）
    is_unique BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 关联关系表
CREATE TABLE relation_definitions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    source_entity_id UUID REFERENCES entity_models(id) ON DELETE CASCADE,
    target_entity_id UUID REFERENCES entity_models(id) ON DELETE CASCADE,
    relation_type VARCHAR(20) NOT NULL, -- one-to-one, one-to-many, many-to-many
    source_field VARCHAR(100),
    target_field VARCHAR(100),
    foreign_key_field VARCHAR(100),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

CREATE TABLE form_fields (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),

    field_label VARCHAR(200),
    field_type VARCHAR(50) NOT NULL, -- 组件类型
    properties JSONB, -- 组件属性
    validation JSONB, -- 验证规则
    visibility_rules JSONB, -- 可见性规则
    default_value TEXT,
    sort_order INTEGER DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 表单实例表
CREATE TABLE form_instances (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),

    tenant_id UUID REFERENCES tenants(id),
    created_by UUID REFERENCES users(id),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    submitted_at TIMESTAMP
);

-- 表单操作日志表
CREATE TABLE form_audit_logs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    form_instance_id UUID REFERENCES form_instances(id) ON DELETE CASCADE,
    operation VARCHAR(50) NOT NULL, -- create, update, submit, approve等

    new_values JSONB,
    operated_by UUID REFERENCES users(id),
    operated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    user_agent TEXT

-- 工作流定义表
CREATE TABLE workflow_definitions (

CREATE TABLE workflow_tasks (

```yaml

Headers: Authorization: Bearer {token}
Response: { success: true }

# 获取当前用户信息
GET /api/auth/me
Headers: Authorization: Bearer {token}
Response: { userId, username, email, fullName, roles, permissions }

# 修改密码
PUT /api/auth/password
Headers: Authorization: Bearer {token}
Request: { oldPassword, newPassword }
Response: { success: true }

# 忘记密码
POST /api/auth/forgot-password
Request: { email }
Response: { success: true }

# 重置密码
POST /api/auth/reset-password
Request: { token, newPassword }
Response: { success: true }
```
