## 📋 项目概述

### 项目名称
**Tianyou** - 企业级低代码开发平台

### 项目目标
开发与某云苍穹低代码平台功能**完全一致**的企业级低代码平台，提供**更人性化**的设计。

### 核心要求
1. ✅ 功能完全一致 - 云苍穹有的功能，Tianyou也要有
2. ✅ 更人性化设计 - 优化用户体验
3. ✅ 技术栈先进 - .NET 8 + Blazor
4. ✅ 隐私保护 - 不出现任何第三方厂商名称
5. ✅ **分阶段交付** - MVP核心功能优先

---

## 💻 技术栈（已调整）

### 后端
- **.NET 8 LTS** - 长期支持版本（稳定）
- **PostgreSQL 15+** - 主数据库
- **Redis 7+** - 缓存
- **RabbitMQ 3.12+** - 消息队列
- **Entity Framework Core 8** - ORM
- **MediatR** - CQRS模式

### 前端
- **Blazor WebAssembly** - Web端
- **Blazor MAUI** - 桌面端（Phase 2）
- **Blazor Mobile** - 移动端（Phase 3）
- **MudBlazor** - Material Design组件库

### DevOps
- **Docker** - 容器化
- **Kubernetes** - 容器编排
- **GitHub Actions** - CI/CD

## 🎯 分阶段交付计划（已调整）

### Phase 1: MVP核心功能（90-120天 / 3.5个月）

**目标**: 可用的低代码平台核心功能

**必须完成**:
- [x] 用户认证授权（JWT + RBAC）
- [x] 单据建模（基础字段类型：文本、数字、日期、选择器）
- [x] 数据模型编辑器（实体建模、字段定义、关联关系）
- [x] 表单设计器（基础组件：输入框、选择器、日期选择器）
- [x] 列表页面（基础过滤、排序、分页）
- [x] REST API（核心CRUD接口）
- [x] 基础插件系统（表单插件基类）
- [x] Web端部署

**暂不实现**:
- ❌ AI功能（智能提示、代码补全）
- ❌ 多人协作编辑
- ❌ 桌面端/移动端
- ❌ 高级报表
- ❌ 工作流引擎（Phase 2）

**成功标准**:
1. ✅ 用户可以注册登录
2. ✅ 可以创建单据模型
3. ✅ 可以设计表单
4. ✅ 可以提交表单数据
5. ✅ 可以列表查看数据
6. ✅ REST API完整可用
7. ✅ 系统可以成功部署到Web端

---

### Phase 2: 高级功能（30天 / 1个月）

**目标**: 完善核心功能，提升易用性

**必须完成**:

- [x] 报表系统（列表报表、分组报表、图表报表）
- [x] 数据集成（API集成、数据库集成）
- [x] 高级字段类型（附件、图片、富文本）
- [x] 高级表单组件（表格、树形选择器）
- [x] 插件系统完善（列表插件、操作插件）
- [x] 更人性化UI（深色模式、快捷键）

**成功标准**:
1. ✅ 工作流引擎可用

3. ✅ 数据集成功能可用
4. ✅ UI更人性化
