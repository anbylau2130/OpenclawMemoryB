## 📋 项目概述

### 项目名称
**Tianyou** - 企业级低代码开发平台

### 项目目标
开发与某云苍穹低代码平台功能**完全一致**的企业级低代码平台，并提供**更人性化**的设计。

### 核心要求
1. ✅ **功能完全一致** - 云苍穹有的功能，Tianyou也要有
2. ✅ **更人性化设计** - 优化用户体验，提升易用性
3. ✅ **技术栈先进** - .NET Core 10 + PostgreSQL + Blazor
4. ✅ **隐私保护** - 不出现任何第三方厂商名称

---

## 💻 技术栈

### 后端
- **.NET Core 10** - 最新版本
- **PostgreSQL 15+** - 主数据库
- **Redis 7+** - 缓存
- **RabbitMQ 3.12+** - 消息队列
- **Entity Framework Core 10** - ORM
- **MediatR** - CQRS模式

### 前端
- **Blazor WebAssembly** - Web端
- **Blazor MAUI** - 桌面端（Windows/macOS/Linux）
- **Blazor Mobile** - 移动端（iOS/Android）
- **MudBlazor** - Material Design组件库

### DevOps
- **Docker** - 容器化
- **Kubernetes** - 容器编排
- **GitHub Actions** - CI/CD
