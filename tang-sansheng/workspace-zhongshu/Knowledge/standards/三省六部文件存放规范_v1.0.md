## ⚠️ 严格规定

### ❌ 禁止行为

1. **❌ 禁止在根目录创建任何文件**
   - 根目录只保留7个核心配置文件
   - AGENTS.md, SOUL.md, TOOLS.md, USER.md, IDENTITY.md, HEARTBEAT.md, README.md

2. **❌ 禁止随意命名文件**
   - 必须使用日期前缀：`YYYYMMDD_`
   - 必须使用中文或英文描述性名称
   - 禁止使用"新建文档"、"未命名"等

3. **❌ 禁止创建深层嵌套**
   - 最多3层子目录
   - 禁止：`docs/reports/2024/03/24/draft/temp/`

4. **❌ 禁止在临时目录存储重要文件**
   - temp/ 会定期清理
   - cache/ 会定期清理

### 5. 💬 对话记录 → `docs/Conversation/`

**⚠️ 重要要求（2026-03-24 09:39）**:
**所有部门必须记录陛下私聊对话！**

**适用**:
- ✅ **陛下私聊对话**
- ❌ 群聊（不记录）

**存储位置**:
```
workspace-[部门]/docs/Conversation/
```

**文件命名**:
```
YYYYMMDD.md

示例：
20260324.md
20260325.md
20260326.md
```

**记录内容**:
- ✅ **只保存陛下说的内容**
- ❌ 不保存自己的回复
- ❌ 不保存其他人的内容

**格式规范**:
```markdown
# [日期] 陛下对话记录

## 09:30
陛下：还有个需求，你需要通知所有agents...

## 09:35
陛下：门下省和尚书省还没有回复呢

## 09:39
陛下：还有个需求，你需要通知所有agents，让他们记录我的每次对话...
```

**示例**:
```markdown
# 2026-03-24 陛下对话记录

## 08:23
陛下：但是其他的部门没有回复给我

## 08:28
陛下：为什么没有权限，你的配置不是跟https://github.com/wanikua/danghuangshang上的一致么？

## 08:36
陛下：告诉所有的三省六部agent文件/root/.openclaw/tang-sansheng/CLAUDE.md是我们的开发规范...

## 09:25
陛下：按你给的文档，整理你的工作目录把，太乱了

## 09:30
陛下：为什么其他六部的成员没有收到文件存放规范的通知？

## 09:39
陛下：还有个需求，你需要通知所有agents，让他们记录我的每次对话(只保留私聊)，并且保存到docs/Conversation中，每天的对话存放一个md文件，只保存我说的内容。
```

**触发条件**:
- ✅ 陛下发送私聊消息时
- ✅ 每次对话后立即追加
- ✅ 当天所有对话存到当天文件

**注意事项**:
- ❌ **不记录群聊**
- ❌ **不保存自己的回复**
- ✅ **只保存陛下内容**
- ✅ **时间精确到分钟**
- ✅ **当天对话追加到当天文件**

**判断私聊方法**:
```javascript
// 检查 chat_type 字段
if (inbound_meta.chat_type === "direct") {
  // 这是私聊，需要记录
  saveConversation(user_message);
}
```

**示例代码**:
```javascript
// 保存对话记录
function saveConversation(message, timestamp) {
  const date = new Date(timestamp);
  const filename = `docs/Conversation/${formatDate(date, 'YYYYMMDD')}.md`;
  const time = formatDate(date, 'HH:mm');
  
  const content = `\n## ${time}\n陛下：${message}\n`;
  
  appendFileSync(filename, content);
}
```

**必须遵守！**
