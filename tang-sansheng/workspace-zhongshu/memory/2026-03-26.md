   - curl -X POST http://localhost:8091/api/configuration/reset
   - 适合配置错误需要重新配置

4. **方式四：通过配置文件手动启用**
   - 编辑 appsettings.json，添加 ConfigurationWizard.Enabled = true
   - 适合高级用户、自动化部署

5. **方式五：通过命令行参数启用**
   - ./Tianyou.Api --force-configuration-wizard
   - 适合临时启用、测试环境
