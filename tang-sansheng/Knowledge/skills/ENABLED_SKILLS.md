## 使用方法

### 为每个Agent启用Skills

每个agent的SOUL.md中添加：

```markdown
## 可用技能
- self-improving-agent
- TaskMaster
- context7
- [其他相关技能...]
```

### 技能调用

在工作中根据需要调用相应技能：
- 任务管理 → TaskMaster
- 代码审查 → code-review
- 加入办公室 → join-office-skill
- 自我优化 → self-improving-agent
