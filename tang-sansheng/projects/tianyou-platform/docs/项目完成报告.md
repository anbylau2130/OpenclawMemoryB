## 🏗️ 系统架构

### 技术栈

#### 后端
- **框架**: .NET 8 LTS
- **数据库**: SQLite（可迁移到PostgreSQL）
- **认证**: JWT Token + BCrypt
- **架构**: DDD四层架构
  - Tianyou.Api - Web API层
  - Tianyou.Application - 应用服务层
  - Tianyou.Domain - 领域模型层
  - Tianyou.Infrastructure - 基础设施层

#### 前端
- **框架**: Blazor WebAssembly
- **样式**: Bootstrap 5
- **状态管理**: Blazor组件状态
- **路由**: Blazor Router

#### 部署
- **Web服务器**: Nginx 1.22
- **反向代理**: Nginx反向代理
- **端口**: 8091

## 🔧 API使用示例

### 用户认证

#### 注册
```bash
curl -X POST http://192.168.50.251:8091/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "testuser",
    "email": "test@example.com",
    "password": "Test@123",
    "fullName": "测试用户"
  }'
```

#### 登录
```bash
curl -X POST http://192.168.50.251:8091/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "testuser",
    "password": "Test@123"
  }'
```

### 数据管理

#### 创建实体
```bash
curl -X POST http://192.168.50.251:8091/api/entities \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer {token}" \
  -d '{
    "entityName": "产品",
    "tableName": "products",
    "description": "产品信息表"
  }'
```

#### 添加字段
```bash
curl -X POST http://192.168.50.251:8091/api/entities/{entityId}/fields \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer {token}" \
  -d '{
    "fieldName": "ProductName",
    "fieldType": "text",
    "isRequired": true,
    "maxLength": 100
  }'
```
