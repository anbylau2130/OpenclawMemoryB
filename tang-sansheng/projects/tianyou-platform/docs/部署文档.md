## 🚀 部署步骤

### 1. 环境准备

```bash
# 安装.NET 8 Runtime
wget https://dot.net/v1/dotnet-install.sh -O dotnet-install.sh
chmod +x dotnet-install.sh
./dotnet-install.sh --channel 8.0 --runtime aspnetcore

# 安装Nginx
apt-get update
apt-get install -y nginx
```

### 2. 部署后端API

```bash
# 创建部署目录
mkdir -p /var/www/tianyou-api

# 复制发布文件
cp -r /root/.openclaw/tang-sansheng/projects/tianyou-platform/backend/src/Tianyou.Api/bin/Release/net8.0/publish/* /var/www/tianyou-api/

# 启动API服务
cd /var/www/tianyou-api
nohup /root/.dotnet/dotnet Tianyou.Api.dll --urls "http://0.0.0.0:5000" > /var/log/tianyou-api.log 2>&1 &
```

### 3. 部署前端Web

```bash
# 创建部署目录
mkdir -p /var/www/tianyou-web

# 复制发布文件
cp -r /root/.openclaw/tang-sansheng/projects/tianyou-platform/frontend/Tianyou.Web/bin/Release/net8.0/wwwroot/* /var/www/tianyou-web/
```

### 4. 配置Nginx

```bash
# 创建Nginx配置
cat > /etc/nginx/sites-available/tianyou << 'EOF'
server {
    listen 8091;
    server_name _;
    
    # 前端静态文件
    location / {
        root /var/www/tianyou-web;
        try_files $uri $uri/ /index.html;
        index index.html;
    }
    
    # API反向代理
    location /api/ {
        proxy_pass http://localhost:5000/api/;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection keep-alive;
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
    
    # 健康检查
    location /health {
        proxy_pass http://localhost:5000/health;
    }
}
EOF

# 启用配置
ln -sf /etc/nginx/sites-available/tianyou /etc/nginx/sites-enabled/tianyou

# 测试配置
nginx -t

# 重启Nginx
systemctl restart nginx || nginx
```

### 5. 验证部署

```bash
# 测试API
curl http://localhost:8091/health

# 测试前端
curl -I http://localhost:8091/

# 测试注册接口
curl -X POST http://localhost:8091/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"test","email":"test@example.com","password":"Test@123","fullName":"测试用户"}'
```

## 🔧 服务管理

### 启动服务

```bash
# 启动API服务
cd /var/www/tianyou-api
nohup /root/.dotnet/dotnet Tianyou.Api.dll --urls "http://0.0.0.0:5000" > /var/log/tianyou-api.log 2>&1 &

# 启动Nginx
nginx
```

### 停止服务

```bash
# 停止API服务
pkill -f "Tianyou.Api.dll"

# 停止Nginx
nginx -s stop
```

### 重启服务

```bash
# 重启API服务
pkill -f "Tianyou.Api.dll"
sleep 2
cd /var/www/tianyou-api
nohup /root/.dotnet/dotnet Tianyou.Api.dll --urls "http://0.0.0.0:5000" > /var/log/tianyou-api.log 2>&1 &

# 重启Nginx
nginx -s reload
```

### 查看日志

```bash
# API日志
tail -f /var/log/tianyou-api.log

# Nginx访问日志
tail -f /var/log/nginx/access.log

# Nginx错误日志
tail -f /var/log/nginx/error.log
```

## 🔒 安全配置

### 1. 防火墙配置

```bash
# 开放8091端口
ufw allow 8091/tcp

# 开放5000端口（内部）
ufw allow 5000/tcp

# 启用防火墙
ufw enable
```

### 2. SSL配置（可选）

```bash
# 安装Certbot
apt-get install -y certbot python3-certbot-nginx

# 获取SSL证书
certbot --nginx -d yourdomain.com

# 自动续期
certbot renew --dry-run
```

### 3. 安全加固

```bash

systemctl disable bluetooth
systemctl disable cups

apt-get update && apt-get upgrade -y
```
