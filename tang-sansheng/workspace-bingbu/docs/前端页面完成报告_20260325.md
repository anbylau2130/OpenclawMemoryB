## 🔌 API接口对应关系

### EntityController
- `GET api/entities/{entityId}` → EntityDetail.razor
- `GET api/entities/{entityId}/data` → EntityData.razor
- `POST api/entities/{entityId}/data` → EntityData.razor
- `PUT api/entities/data/{dataId}` → EntityData.razor
- `DELETE api/entities/data/{dataId}` → EntityData.razor

### FormController
- `GET api/forms` → FormList.razor
- `GET api/forms/{formId}` → FormDesigner.razor, FormPreview.razor
- `POST api/forms/{formId}/fields` → FormDesigner.razor
- `POST api/forms/{formId}/publish` → FormDesigner.razor

### WorkflowController

- `GET api/workflows/my-tasks` → WorkflowTasks.razor
