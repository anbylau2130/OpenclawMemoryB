### 数据模块实现

```python
import pandas as pd
import tushare as ts
import akshare as ak

class DataModule:
    def __init__(self, source='akshare'):
        self.source = source
        
        if source == 'tushare':
            ts.set_token('your_token')
            self.pro = ts.pro_api()
    
    def get_daily_data(self, symbol, start_date, end_date):
        """获取日线数据"""
        if self.source == 'tushare':
            df = self.pro.daily(
                ts_code=symbol,
                start_date=start_date,
                end_date=end_date
            )
        
        elif self.source == 'akshare':
            df = ak.stock_zh_a_hist(
                symbol=symbol,
                period='daily',
                start_date=start_date,
                end_date=end_date,
                adjust='qfq'  # 前复权
            )
        
        return df
```

- AKShare被服务器限制，无法批量获取
- 解决方案：使用requests直接调用新浪/东财API
- 缓存机制避免重复请求

- 模拟数据无法反映真实市场
- 回测胜率45.9%不可靠
- 必须使用真实数据验证

### 3. 策略迭代方法
- 先分析已有回测数据，找出高胜率因子
- 移除低效因子，不追求复杂
- 盈亏比比胜率更重要
